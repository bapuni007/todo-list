# 📋 TODO List App

The **TODO List App** is a simple web application built with **HTML, CSS (Bootstrap)**, and **JavaScript** that helps users manage daily tasks efficiently. It provides a clean interface to add, view, and delete items, with persistent storage using the browser’s **localStorage**.

---

## ✨ Features
- Add tasks with a **title** and **description**
- Display tasks dynamically in a **Bootstrap table**
- Delete individual tasks with a single click
- Clear all tasks at once using the **Clear List** button
- Prevents saving empty or space‑only tasks
- Automatically resets input fields after adding a task
- Data persists across page reloads using **localStorage**

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Bootstrap 4  
- **Logic:** Vanilla JavaScript  
- **Storage:** Browser localStorage
